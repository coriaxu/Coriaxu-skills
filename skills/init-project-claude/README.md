# init-project

## 目的

这个 skill 用来在项目创建时一次性把记忆系统搭好，重点是让 `memory_map.md` 从第一天就存在，避免后续再补路由层。

## 2026-03-06 本次升级

- 新项目默认创建 `memory_map.md`
- 记忆模板统一使用中文
- 默认场景改成 L&D 高频工作场景
- 对 Claude 明确了 `Glob`、`Read`、`Write`、`Edit`、`Bash` 的边界

## 后续建议

- 创建项目后，第一时间补齐“项目目标”和“当前主要产出”
- 如果项目并不符合默认 L&D 场景，应尽早在 `memory_map.md` 中调整路由
