## 功能

结构化圆桌讨论器。围绕一个议题邀请 3–5 位真实历史或当代人物展开多轮辩证讨论，由追求真相的主持人持续提炼核心争议、生成 ASCII 思考框架图，并最终输出知识网络。

触发词：圆桌讨论、圆桌、roundtable、辩论。
