## 功能

内容铸造器 · Content Caster。将内容铸成 PNG 视觉图片，六种模具：长阅读卡（`-l`）、信息图（`-i`）、多卡片（`-m`）、视觉笔记（`-v`）、漫画（`-c`）、白板（`-w`）。输出至 `~/Downloads/`。

触发词：铸、cast、做成图、做成卡片、做成信息图、做成海报、视觉笔记、sketchnote、漫画、comic、白板、whiteboard。
