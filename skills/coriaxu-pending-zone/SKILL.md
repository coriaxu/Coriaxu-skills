# 【Coriaxu】Pending Zone

这是一个通用版个人悬置区 skill。它把“值得保留，但还没想清楚”的议题放进桌面文件夹，避免它们混进待办清单，也避免被忘掉。

## 默认存储位置

```bash
$HOME/Desktop/个人悬置区/
```

首次使用时自动创建目录。活跃卡片放在根目录，已归档卡片放在：

```bash
$HOME/Desktop/个人悬置区/归档/
```

## 三个动作

- 建卡：把未决议题保存为 `YYYY-MM-DD-主题.md`
- 查询：列出当前仍处于 `悬置中` 的卡片
- 归档：把已有结论的卡片改状态并移入 `归档/`

详细执行步骤见 [references/runbook.md](references/runbook.md)，卡片格式见 [references/card-template.md](references/card-template.md)，边界判断见 [references/boundary.md](references/boundary.md)。

## 核心规则

- “悬置”是显式命令，出现后直接建卡。
- “先放一放”“回头再说”等表达只算嗅探信号，必须先问用户是否要建卡。
- 日期必须用 `date +%Y-%m-%d` 获取。
- 不把纯待办、项目任务、长期偏好写进悬置区。
- 已归档卡片默认不再修改，除非用户明确要求。
