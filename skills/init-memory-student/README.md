# Init Memory Student

## 适用场景

这个技能适合已经有项目文件夹、但还没有给 AI 建“记忆区”的场景。常见文件会是 `PPT`、`Excel`、`Word`、`PDF`、图片这些业务材料。它会先看文件夹名和文件名，尽量用最少的信息理解项目，再把准备记住的内容先展示给你看。

## 不会做什么

它不会新建项目，也不会改动你原来的业务文件。你没有点头之前，它不会创建任何记忆文件。它也不会去通读整堆材料，只会在实在判断不出来时，补看 1 到 2 个最像“项目说明”的文件开头。

## 课堂使用示例

把终端切到你的项目文件夹里，输入 `/init-memory`。如果 AI 理解对了，你会先看到它对项目的判断，以及准备记住的几段内容。你确认后，它才会在项目旁边生成 `AI_专属` 文件夹，方便下次继续接着干。
