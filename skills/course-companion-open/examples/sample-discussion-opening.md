# Sample Discussion Opening

```md
我先用 `three-level-notes` 拆完了骨架。这堂课的核心不是“怎么做笔记”，而是一个更深的问题：知识什么时候才真的进入一个人的行动系统。

这次你选择的是无知识库模式，所以我不扫描旧笔记。我先抓 3 个课程内部锚点：

1. **信息不等于改变**：课程把“知道”和“行动变化”切开了。
2. **场景是知识的入口**：知识要进入任务、关系、选择和反馈，才会产生力量。
3. **学习要找到真实问题**：如果找不到对应问题，课程就停留在摘要层。

我的真实反应是：这堂课最有价值的地方，是它把学习从“内容处理”推回了“生活中的调用”。但它还有一个没讲透的地方：不是每个人都能立刻说清自己的真实问题。有时候人是先被一句话击中，过几天才知道它对应什么。

我想先问你一个问题：这堂课里哪句话最像是在说你最近的某个具体处境？
```
